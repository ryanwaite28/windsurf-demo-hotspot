# Changelog: 2026-02-27 17:30 EST

## Summary

Initial backend scaffold and implementation of all MVP specs (User Accounts, Social Zone) for the HotSpot platform.

## Specs Implemented

### User Accounts (Backend)
- **Entities**: User, UserPrivacySettings, UserSecuritySettings, UserDevice, UserFollow, UserMute, UserBlock, UserReport
- **Auth**: JWT-based signup, login, logout, token refresh, password reset flow
- **Profile**: CRUD, privacy settings, security settings
- **Social interactions**: follow/unfollow, mute/unmute, block/unblock, report
- **Tests**: 19 unit tests (AuthServiceTest, UserServiceTest, UserSocialServiceTest)

### Social Zone (Backend)
- **Entities**: Thread, ThreadMedia, ThreadPoll, ThreadPollOption, ThreadPollVote, ThreadLike, ThreadReaction, ThreadComment, ThreadSave, ThreadReport, ThreadMute, CommentReport, CommentMute
- **Threads**: CRUD, repost, quote, pin, 10-minute edit window, media attachments, polls
- **Interactions**: like/unlike, react, save/bookmark, report, mute
- **Comments**: CRUD with 10-minute edit window, report, mute
- **Feed**: Reverse-chronological feed from followed users
- **Tests**: 17 unit tests (ThreadServiceTest, CommentServiceTest)

## Infrastructure

- **Spring Boot 3.2.3** with Java 21, Spring Data JPA, Spring Security
- **Database**: PostgreSQL 16 with Flyway migrations (V1__init_schema.sql)
- **Cache**: Redis 7
- **Security**: JWT access/refresh tokens, BCrypt password hashing
- **Docker Compose**: Full stack (backend + deps) and deps-only configurations
- **LocalStack**: S3, SQS, SES, SNS with auto-provisioning init script
- **MailHog**: Local email testing (SMTP on 1025, Web UI on 8025)
- **Dockerfile**: Multi-stage build using Maven wrapper
