# Requirements: HotSpot
Date: 2026-02-27
Time: 13:17 EST


## Functional Requirements

- User Accounts
    - Users have profiles
        - Email
        - Phone (optional, hidden/visible via settings)
        - Display Name
        - Username
        - Bio
        - Profile Picture
        - Location (optional, hidden/visible via settings)
    - Users can signup (email, display name, username, password, confirm password)
    - Users can login (email, password)
    - Users can logout
    - Users can update their profile settings
        - Display Name
        - Username
        - Bio
        - Profile Picture
        - Location
        - Privacy Settings
            - Who can see user's profile
            - Who can see user's posts
            - Who can see user's location
            - Who can see user's phone number
        - Security Settings
            - 2FA
            - Devices (list of devices that have logged in)
    - Users can delete their account
    - Users can reset their password
    - Users can add/change/verify their phone number
    - Users can change their email
    - Users can follow/unfollow other users
    - Users can mute/unmute other users
    - Users can block/unblock other users
    - Users can report other users

- Zones
    - Social Zone
        - Users can create/repost/quote posts (called "threads") in the same way how Twitter/Threads works
        - Users can like/unlike threads
        - Users can react to threads (heart, laugh, etc)
        - Users can comment on threads
        - Users can share threads
        - Users can save threads
        - Users can report threads
        - Users can delete threads
        - Users can pin threads
        - Users can mute threads
        - Users can edit threads within 10 minutes of creation
        - Users can report comments
        - Users can delete comments
        - Users can mute comments
        - Users can edit comments within 10 minutes of creation
        - Users can add media (images, videos, audio) to threads
        - Users can add polls to threads
        - Users can add voice notes to threads
    - Radio Zone
    - Livestream Zone


## Non-Functional Requirements

- Infrastructure
    - The app must be hosted on AWS
- Availability
    - The app must be available 24/7
    - The app must be available in the US in a single region for MVP
- Performance
    - The app must be able to handle 1000 users per second for MVP
